# Genealogy Tree API

## Описание
Система генеалогического дерева с проверкой целостности данных и конфиденциальностью.

## Требования
- Управление персонами (ФИО, пол, даты, биография)
- 3 типа связей: супруги, родитель-ребенок, братья-сестры
- Конфиденциальность: пользователи видят только свои данные
- Поиск родственников (дети, родители, братья/сестры, предки, потомки)
- Проверка непротиворечивости данных

## Технологии
- .NET 9.0, ASP.NET Core Web API
- PostgreSQL, Entity Framework Core
- JWT аутентификация
- NUnit для тестирования
- Swagger для документации

## Установка
1. Клонировать репозиторий
2. Установить PostgreSQL
3. Настроить строку подключения в `appsettings.Development.json`
4. `dotnet ef database update`
5. `dotnet run`

## API
- `POST /api/auth/register` - регистрация
- `POST /api/auth/login` - вход
- `GET /api/persons` - список персон
- `POST /api/relationships` - создание связи
- `GET /api/relationships/person/{id}` - связи персоны
